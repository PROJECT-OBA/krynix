# Git Workflow

## Branching Model

Protected branch:
- main

Feature branches:
- feat/*
- fix/*
- chore/*
- docs/*

## Development Flow

1. Pull latest main
2. Create branch
3. Commit small logical units
4. Push branch
5. Open PR
6. Wait for CI
7. Squash merge

## Hotfixes

Create fix/* branch.
Follow normal PR process.
